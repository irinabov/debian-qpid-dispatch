qdrouterd manual page
=====================

SYNOPSIS
--------

qdrouterd [*options*]

DESCRIPTION
-----------

The Qpid Dispatch router (qdrouterd) is a network daemon that directs
AMQP 1.0 messages between endpoints, such as messaging clients and
servers.
